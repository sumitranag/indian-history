# indian-history

A simple frontend web app that lets users ask questions related to Indian history using OpenAI's GPT model.

## Features
- Built using HTML, CSS, and JavaScript
- Uses OpenAI GPT API
- Simple and mobile-friendly UI

## Setup
1. Clone this repo
2. Replace `YOUR_OPENAI_API_KEY` in `script.js` with your key
3. Open `index.html` in your browser

## Note
For production, consider using a backend or secure environment for your API key.
